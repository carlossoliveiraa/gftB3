# API de Investimento CDB

## Descrição
Esta API calcula o retorno de investimentos em Certificados de Depósito Bancário (CDB), 
considerando diferentes prazos e a incidência de impostos conforme a duração do investimento.

## Tecnologias Utilizadas
- .NET Framework 4.7.2
- MSTest para realização dos testes unitários
